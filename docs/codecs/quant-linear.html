quant_linear

Uniform steps, bounded error.

Near-lossless quantization on a uniform grid. It rounds every value to the nearest multiple of a step, q = round(x / scale) with scale = 2·max_error, so each reconstruction lands within ±max_error of the original. What travels is the small integer index q, which the stages downstream compress far better than the raw value. A near-lossless codec sits once at the head of the graph, so the loss happens a single time and every stage after it stays exact.
