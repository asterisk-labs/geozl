average

The floor average of W and N.

The gentlest 2D predictor. Each pixel is predicted by the floor average of its west and north neighbours, floor((W + N) / 2), computed as (W>>1) + (N>>1) + (W & N & 1) so it never overflows the sample width. Edge neighbours are zero, so the first row predicts floor(W/2) and the first column floor(N/2). It reads the reconstructed W, so each sample is decoded in order. A good fit for smooth, slightly noisy tiles where neither a delta nor a full plane wins.
