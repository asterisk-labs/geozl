planar

The plane through W, N and NW.

A true two-dimensional predictor. Each pixel is predicted by the plane passing through three neighbours the decoder already holds, W + N − NW. Row zero falls back to the horizontal predictor, column zero to the vertical one. The prediction carries no left-dependency beyond W, so a row still decodes as a fast prefix sum. On any locally planar gradient the residuals collapse to zero.
