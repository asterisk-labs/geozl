delta_n

Each value, minus its north neighbour.

The vertical twin of delta_w. Every sample is stored as the difference from the one directly above it, and the first row is kept whole as a seed. Columns are independent, with no horizontal carry, so the transform shines on smooth vertical gradients, where each residual collapses toward zero for the entropy stage.
